使用指南
什么是整体主题
整体主题用于完全替换整个聊天详情页面的CSS样式，实现统一的视觉效果。与分组主题（顶栏、底栏、气泡）不同，整体主题的CSS作用于整个页面。
优先级规则
整体主题优先级最高
整体主题 > 分组主题（顶栏/底栏/气泡）

启用整体主题后，分组主题仍会生效，但可能被整体主题的样式覆盖。
目标选择器
#chatDetailScreen 聊天详情页容器
.chat-detail-header 顶栏区域
.chat-messages-area 消息区域
.chat-input-area 底栏区域
.chat-message-bubble 消息气泡
可用变量
整体主题支持所有变量：
颜色、头像、尺寸等，与其他主题类型相同
常见问题
Q1: 整体主题和分组主题冲突？
原因：整体主题和分组主题同时启用时，样式可能重叠。

解决：使用 !important 提高整体主题的优先级，或禁用分组主题。
Q2: 如何实现整页背景？
#chatDetailScreen {
  background: linear-gradient(180deg, #667eea, #764ba2) !important;
  background-size: cover !important;
}
Q3: 整体主题适合什么场景？
适合：需要统一风格、整页背景、全局色调
不适合：只想修改某个区域（建议用分组主题）
Q4: 整体主题能用HTML注入吗？
可以！整体主题支持HTML自定义装饰，会注入到 #chatDetailScreen 容器内。
Q5: 如何区分多个层级（z-index）？
/* 背景装饰层 */
.custom-bg-layer {
  position: absolute;
  z-index: 0; /* 最底层 */
}

/* 消息区域 */
.chat-messages-area {
  position: relative;
  z-index: 1; /* 中间层 */
}

/* 顶栏/底栏 */
.chat-detail-header,
.chat-input-area {
  z-index: 10; /* 前景层 */
}
Q6: 如何实现融为一体无分割线？
/* 统一背景色 */
#chatDetailScreen,
.chat-detail-header,
.chat-messages-area,
.chat-input-area {
  background: {{USER_BG}} !important;
}

/* 移除所有边框和阴影 */
.chat-detail-header,
.chat-input-area {
  border: none !important;
  box-shadow: none !important;
}
Q7: 如何实现玻璃态毛玻璃效果？
.chat-detail-header,
.chat-input-area {
  background: rgba(255,255,255,0.1) !important;
  backdrop-filter: blur(10px) !important;
  border: 1px solid rgba(255,255,255,0.2) !important;
}

我需要为一个聊天应用生成整体主题。

【系统说明】
- 这是一个聊天详情页主题编辑器
- 支持CSS样式和HTML装饰元素
- CSS会自动添加 #chatDetailScreen 前缀作用域
- HTML会被注入到对应区域

【可用变量（12个）】

尺寸类：{{BUBBLE_MAX_WIDTH}} {{BUBBLE_FONT_SIZE}} {{BUBBLE_BORDER_RADIUS}} {{BUBBLE_PADDING}}

【主要选择器】
.chat-detail-header（顶栏）
.chat-messages-area（消息区）
.chat-input-area（底栏）
.chat-message-bubble.user（用户气泡）
.chat-message-bubble.character（角色气泡）

【需求描述】
风格：[在此描述主题风格，如"极简黑白灰"、"赛博朋克"等]

顶栏设计：[描述顶栏样式]
消息区背景：[描述整体背景]
底栏设计：[描述输入区样式]
气泡样式：[描述消息气泡外观]

特殊效果：[如渐变、阴影、动画、玻璃态等]

【输出要求】
1. 提供完整CSS代码（不要包含 #chatDetailScreen 前缀，会自动添加）
2. 如需装饰元素，提供三个HTML代码块：
- 顶栏HTML（注入到 .chat-detail-header）
- 底栏HTML（注入到 .chat-input-area）
- 气泡区HTML（注入到 .chat-messages-area）
3. 使用变量而非硬编码颜色（提高可定制性）
4. 确保深色/浅色模式兼容性
示例1：极简黑白灰
风格：极简黑白灰，无彩色，现代感
顶栏：纯白背景，细灰色底边框（1px #e0e0e0）
消息区：浅灰渐变背景（#fafafa到#f5f5f5）
底栏：纯白背景，细灰色顶边框
气泡：用户用深灰色（#333），角色用白色带灰边框（#e0e0e0）
特殊效果：极简设计，无阴影，圆角8px
示例2：赛博朋克
风格：赛博朋克，霓虹色，未来感
顶栏：黑色渐变（#000到#0a0a0a），下边框霓虹蓝发光（box-shadow: 0 1px 10px #00ffff）
消息区：深紫到黑渐变（#1a0033到#000）
底栏：半透明黑（rgba(0,0,0,0.8)），上边框霓虹粉发光（#ff006e）
气泡：用户霓虹粉（#ff006e）带外发光，角色霓虹蓝（#00ffff）带外发光
特殊效果：所有边框2px发光，文字带微弱阴影
HTML装饰：顶栏添加扫描线动画、底栏添加闪烁光标效果
示例3：玻璃态拟物
风格：玻璃态毛玻璃，拟物化，柔和
顶栏/底栏：半透明白（rgba(255,255,255,0.1)），毛玻璃（backdrop-filter: blur(20px)），白色边框（rgba(255,255,255,0.2)）
消息区：使用{{USER_BG}}变量作为整页渐变背景
气泡：用户/角色都用毛玻璃效果，带细微内阴影
特殊效果：所有元素圆角16px，柔和阴影（0 8px 32px rgba(0,0,0,0.1)）
HTML装饰：顶栏添加3个浮动光斑装饰（position: absolute，带动画）
高级技巧
√ 明确变量使用 - 告诉AI"用{{USER_BG}}而非硬编码颜色"
√ 分离CSS和HTML - 要求"CSS和HTML分开输出"
√ 提供参考 - "类似iOS消息"、"仿Discord风格"
√ 分步优化 - 先基础版→再要求添加动画/装饰
√ 兼容性要求 - "确保深色模式下也好看"
× 避免模糊描述 - 不要只说"好看的"、"炫酷的"

常见问题
Q1: 好感度/返回/设置按钮样式无法覆盖？
原因：JavaScript设置了内联样式，优先级比CSS高

解决：系统已自动处理。如果仍有问题，确保使用完整覆盖：
.chat-affection-indicator {
  width: 26px !important;
  height: 26px !important;
  overflow: hidden !important;
  position: relative !important;
}
.chat-affection-indicator > * {
  display: none !important;
}
Q2: 返回按钮大小/边框不对？
原因：原始CSS有多个属性，只覆盖部分会保留其他

解决：使用min/max-width/height强制固定尺寸，显式设置border:none
.chat-detail-back-btn {
  width: 30px !important;
  min-width: 30px !important;
  max-width: 30px !important;
  border: none !important;
  padding: 0 !important;
}
Q3: 按钮图标样式无法修改？
原因：CSS无法修改SVG的path属性，只能改颜色/大小

解决：隐藏原始SVG，用伪元素+SVG data URL显示新图标
/* 返回按钮：隐藏原SVG */
.chat-detail-back-btn svg {
  display: none !important;
}
/* 用伪元素显示简单箭头 */
.chat-detail-back-btn::before {
  content: '' !important;
  width: 16px !important;
  height: 16px !important;
  background-image: url("data:image/svg+xml,...") !important;
  background-size: contain !important;
}
Q4: HTML中的变量没有被替换？
检查：
1. 变量名必须全大写：{{CHAR_NAME}}
2. 确保系统版本为v3.0+（已支持HTML变量替换）
3. 变量两边是双花括号，无空格
Q5: 顶栏过高遮挡聊天内容？
原理：系统会自动计算顶栏高度并调整内容区域

如果自动调整不准确，可手动设置：
.chat-messages-area {
  top: 280px !important; /* 根据顶栏实际高度调整 */
}

Q1: 装饰层(.theme-decor-*)定位不准？
原因：装饰层是.chat-input-area的子元素，定位参照物是整个底栏，不是输入框容器。

解决：改用.chat-input-wrapper的伪元素(::before/::after)，它们定位参照输入框本身更精准。
Q2: 如何在输入框内添加装饰图标？
推荐：使用 inline SVG data URL + 伪元素
.chat-input-wrapper::before {
  content: '';
  position: absolute;
  right: 50px; top: 50%;
  transform: translateY(-50%);
  width: 24px; height: 24px;
  background-image: url("data:image/svg+xml,...");
  pointer-events: none;
}
Q3: 装饰图标和文字重叠？
原因：输入框文字区域没有为图标预留空间。

解决：给输入框添加右边距
.chat-message-input {
  margin-right: 70px !important; /* 为图标留空间 */
}
Q4: 替换按钮图标的技巧？
1. 隐藏原SVG: 按钮 svg { display: none; }
2. 用::before/::after画新图标
3. 按钮需设置 position: relative
/* 相机图标示例 */
.chat-attach-btn::before { /* 机身 */
  content: ''; position: absolute;
  width: 18px; height: 14px;
  border: 2.5px solid #fff;
  border-radius: 3px;
}
.chat-attach-btn::after { /* 镜头 */
  content: ''; position: absolute;
  width: 7px; height: 7px;
  border: 2px solid #fff;
  border-radius: 50%;
}
Q5: 发送按钮放在输入框内部？
使用绝对定位将发送按钮放在输入框容器内：
.chat-send-btn {
  position: absolute !important;
  right: 8px !important;
  top: 50% !important;
  transform: translateY(-50%) !important;
}
